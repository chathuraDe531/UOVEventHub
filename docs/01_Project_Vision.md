# UOVEventHub

## Project Vision

UOVEventHub is a centralized event management system developed for the University of Vavuniya. The platform enables students to discover upcoming events, register for events, view previous event reports, and stay informed about university activities.

The system also provides organizations such as student unions, clubs, societies, faculties, and departments with a dedicated platform to create, manage, and promote events. It includes event organization guidelines, approval procedures, document templates, and event management tools to simplify the event planning process.

The goal of UOVEventHub is to replace scattered communication methods with a single digital platform that improves event management, participation, and collaboration across the University of Vavuniya.

---

## Objectives

- Centralize all university events into one platform.
- Provide students with easy access to upcoming events.
- Allow organizations to manage events efficiently.
- Provide event organization guidelines and approval procedures.
- Maintain reports and summaries of completed events.
- Improve communication between organizers and students.
- Create a digital archive of university events.

---

## Target Users

### Students
- Browse upcoming events.
- Register for events.
- View previous event reports.
- Receive announcements and notifications.

### Organizations
- Student Unions
- Clubs
- Societies
- Faculties
- Departments

Organizations can create, edit, publish, and manage university events.

### Administrator
- Manage users.
- Manage organizations.
- Manage event categories.
- Monitor system activities.

---

## Scope

The first version (MVP) of UOVEventHub will include:

- User authentication
- Organization dashboard
- Event creation
- Event registration
- Event reports
- Event gallery
- Event organization guidelines
- Notification system

Future versions will include:

- QR attendance
- Certificate generation
- Email notifications
- Resource management
- Analytics dashboard
